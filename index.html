One-line install scripts.

Shell (zsh + oh-my-zsh + powerlevel10k)

bash -c "$(curl -fsSL https://get.abd.dev/setup-shell.sh)"

Installs zsh (if missing), oh-my-zsh, powerlevel10k, zsh-autosuggestions, zsh-syntax-highlighting, nvm, uv, and the entire CLI. macOS and Ubuntu. Safe to re-run.
